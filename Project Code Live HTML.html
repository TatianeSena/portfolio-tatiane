Projeto 1

Mozilla Firefox

Se você procura um browser completo, o navegador Mozilla Firefox é uma ótima escolha. A Mozilla oferece um navegador rápido, balanceado e repleto de funções, como o modo escuro, bloqueio de rastreadores e a sincronização entre dispositivos, que facilita o uso no dia a dia.

Projeto 2

Google Chrome

O navegador Google Chrome dispensa apresentações, sendo o browser mais utilizado no mundo. Entre seus principais recursos estão o modo escuro e o acesso à sua vasta lista de extensões e temas da Chrome Web Store. Também há a possibilidade de criar usuários diferentes para o navegador.

Projeto 3

Opera

Um dos navegadores mais antigos, o Opera nasceu em 1995 e até hoje é um dos principais navegadores, com milhões de usuários pelo mundo. Conhecido pela inovação, oferece um serviço de VPN gratuito que permite ao usuário acessar conteúdos específicos de outros países.

Projeto 4

Microsoft Edge

O Microsoft Edge é o navegador oficial do Windows e, assim como o Safari no macOS, ele já vem pré-instalado com o sistema operacional. O foco do Microsoft Edge é ser direto ao ponto, sendo uma boa opção para quem procura um navegador leve e simples.
